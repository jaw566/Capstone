F1/10 Yellowtails

Platform to make F1/10 Autonomous Racing easy for rookie coders.

What is F1/10 Autonomous Racing?

F1/10 Autonomous RC cars are 1/10 the size of a formula one car. Reachong top speeds of 40mph and using lidars and sensors you would find on real self driving vehicles, this platform is used to research and test autonomous driving solutions. Currently an extensive knowledge of the Robotic Operating System (ROS) and Linux are required to use these cars.

Through our new cutting edge graphical user interface, new coders will be able to utilize the F1/10 platform to start autonomous racing. By abstracting the Robotic Operating System (ROS) and the comand line interface we lower the entry level of autonomuos racing.

F1/10 Yellowtails

A project that re-defines access to the F1/10 Autonomous Racing platform. Providing an intuitive and easy to use system for rookie coders.

What is F1/10 Autonomous Racing?

F1/10 Autonomous RC cars are 1/10th the size of a formula one car. Reaching top speeds of 40mph and using lidars and sensors you would find on real self driving vehicles, the F1/10 Autonomous Racing platform is used to research and test autonomous driving solutions. Without our project, an extensive knowledge of the Robotic Operating System (ROS) and Linux are required to drive these cars.

Robotic Operating System

Yellowtails Project

The Yellowtails project is a graphical user interface that allows you to drive an F1/10 vehicle autonomously. This improvement of access to autonomous racing means that the Yellowtails project can provide a learning platform to people with limited coding experience.

Through our new cutting edge graphical user interface, new coders will be able to utilize the F1/10 platform to start autonomous racing. By abstracting the Robotic Operating System (ROS) and the comand line interface we lower the entry level of autonomuos racing.
